    Item Numbering

    1 done
    ✓ Completed

    Automatic item-code numbering

    Item codes generate as a type prefix + number (e.g. K0232 — “K” for Kurti). The number auto-increments, you can set a custom number out of sequence, and any number already in use is skipped. The 4-digit limit (9999) is handled: once reached it restarts at 1 and reuses only genuinely free numbers, blocking reuse of numbers still held by active stock until old items are cleaned up.

    Billing & TDS

    1 done
    ✓ Completed

    Branch-wise TDS on sales invoices

    Each billing branch now has a “TDS Registered (0.1%)” setting. When it’s on, sales invoices for that branch automatically deduct 0.1% TDS and show Gross Amount → TDS Deducted → Net Payable (branches without it are unaffected). Amounts round correctly to two decimals.
